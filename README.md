# 🏏 IPL Data Analysis Project

## 📌 Project Overview
This project performs an in-depth analysis of Indian Premier League (IPL) match data using Python.
The objective is to extract meaningful insights from historical IPL data through data cleaning,
exploratory data analysis (EDA), and visualizations.

---

## 🧰 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset Description
- *matches.csv* – Match-level details (season, teams, venue, winner, etc.)
- *deliveries.csv* – Ball-by-ball data for each match

---

## 🧹 Data Cleaning Steps
- Removed irrelevant columns (umpire details)
- Handled missing values in city, winner, and player of match
- Verified data consistency after cleaning

---

## 📊 Analysis & Visualizations
- Top 5 Teams with Most Wins
- Top 5 Batsmen with Highest Runs
- Total Runs Scored per Over
- Total Runs per Over per Match
- Winning Margin (Wickets) Distribution

---

## 📈 Key Insights
- Certain teams dominate consistently across seasons
- Powerplay overs contribute significantly to total runs
- A few batsmen contribute a major share of total runs
- Most matches are won by a margin of 5–7 wickets
